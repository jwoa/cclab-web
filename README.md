# CC Lab Portfolio Slides

An interactive React-based presentation for teaching web development basics including HTML, CSS, and accessibility.

## Features

- 24 slides covering web development fundamentals
- Interactive navigation with keyboard support (arrow keys)
- Progress bar and slide indicators
- Beautiful gradient UI with Tailwind CSS
- Fully accessible with ARIA labels

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running the Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:5173` (or another port if 5173 is in use)

### Building for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Navigation

- **Arrow Keys**: Navigate between slides (Left/Right)
- **Previous/Next Buttons**: Click to navigate
- **Slide Indicators**: Click any dot to jump to a specific slide

## Topics Covered

1. Introduction to Web Development
2. HTML Basics & Structure
3. CSS Styling & Layout
4. Web Accessibility
5. Portfolio Building
6. Best Practices & Resources
